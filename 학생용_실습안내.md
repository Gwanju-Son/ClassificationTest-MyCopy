## 💪 도전 과제

### 초급 과제
1. **NUM_EPOCHS**를 20으로 늘려보세요
   - 정확도가 얼마나 향상되나요?

2. **LEARNING_RATE**를 변경해보세요
   - `0.0001`로 줄이면?
   - `0.01`로 늘리면?

### 중급 과제
3. **은닉층 크기** 변경
   ```python
   model = MLP(input_size=3072, 
               hidden_size1=1024,  # 512 → 1024
               hidden_size2=512)   # 256 → 512
   ```
   - 파라미터 수가 어떻게 변하나요?
   - 성능은 향상되나요?

4. **Dropout 비율** 조정
   - `0.3` → `0.5`로 늘리면?
   - `0.1`로 줄이면?

### 고급 과제
5. **은닉층 추가하기**
   - 3개의 은닉층을 가진 모델을 만들어보세요
   - 깊어진 만큼 성능도 향상되나요?

6. **다른 활성화 함수 시도**
   - ReLU 대신 LeakyReLU를 사용해보세요
   ```python
   self.relu1 = nn.LeakyReLU()
   ```
